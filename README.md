# MediIntel-ai
**MediIntel AI** is an AI-powered healthcare platform that converts unstructured medical documents into structured insights. Using OCR and NLP technologies, it extracts important information such as diseases, medications, dosages, and risk factors from medical records. The system provides an interactive dashboard, patient summaries, risk alerts.
